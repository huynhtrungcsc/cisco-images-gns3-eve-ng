# Security Policy

## Scope

This repository contains only documentation and download links pointing to external file hosting services.  
It does not host, distribute, or execute any software directly.

---

## Reporting a Security Concern

If you discover a security-related issue — such as a link pointing to malicious content, a phishing URL, or a compromised file — please report it **responsibly** before making it public.

### How to report

1. **Do not open a public Issue** for security concerns.
2. Contact the maintainer directly via GitHub:
   - Navigate to the [Security tab](../../security) of this repository.
   - Use **"Report a vulnerability"** (GitHub Private Vulnerability Reporting).
3. Include in your report:
   - The specific link or entry that is affected.
   - A description of the potential risk.
   - Any evidence or reproduction steps, if applicable.

We will acknowledge your report within **72 hours** and aim to resolve confirmed issues promptly.

---

## Disclaimer on Linked Content

All download links in this repository point to third-party hosting services (Google Drive, MEGA, etc.).  
The maintainers of this repository:

- Do **not** control or audit the content hosted on external platforms.
- Cannot guarantee the integrity or safety of externally hosted files over time.
- Recommend always scanning downloaded files with an up-to-date antivirus tool before use.

---

## Supported Versions

This repository does not release versioned software. All content in the `main` branch is considered current.
