## Usage Notes

- All images are `.image` format compatible with **Dynamips** (used by GNS3).
- Recommended RAM allocation per platform:

  | Platform | Recommended RAM |
  |----------|----------------|
  | c1700 | 64–128 MB |
  | c2600 / c2691 | 128–256 MB |
  | c3620 / c3640 / c3660 | 128–256 MB |
  | c3725 / c3745 | 128–256 MB |
  | c7200 | 256–512 MB |

- Idle-PC values should be calculated in GNS3 after loading the image to reduce CPU usage.
