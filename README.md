## Usage Notes

- All images use `.image` format, compatible with **Dynamips** (the engine behind GNS3 IOS routers).
- Recommended RAM allocation per platform:

  | Platform | Recommended RAM |
  |----------|:--------------:|
  | c1700 | 64 – 128 MB |
  | c2600 / c2691 | 128 – 256 MB |
  | c3620 / c3640 / c3660 | 128 – 256 MB |
  | c3725 / c3745 | 128 – 256 MB |
  | c7200 | 256 – 512 MB |

- Always calculate **Idle-PC** values in GNS3 after loading a new image to prevent excessive CPU usage.
